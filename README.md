# WinHub

**WinHub** — автономный центр загрузки ПО для Windows. Один `.exe`, без установки .NET/Python/Node. Работает на чистом Win32 API.

Альтернатива Microsoft Store: каталог популярных бесплатных программ, поиск, фильтр по категориям, загрузка и запуск установщиков.

## Возможности

- **20+ программ** — браузеры, разработка, медиа, утилиты, игры
- **Поиск и категории** — быстро найти нужное
- **Один клик — установка** — скачивание + запуск установщика
- **Автономный exe** — статическая линковка, только системные DLL Windows
- **Тёмный интерфейс** — современный вид
- **Без телеметрии** — ничего никуда не отправляется

## Сборка

### Требования

- **Visual Studio Build Tools** (рекомендуется) или **MinGW-w64**
- Windows 10/11

### Команда

```bat
build.bat
```

Результат: `build\WinHub.exe`

### Ручная сборка (MSVC)

```bat
cl /EHsc /O2 /MT /DUNICODE /D_UNICODE /Fe:WinHub.exe src\main.cpp src\download.cpp ^
   user32.lib gdi32.lib comctl32.lib winhttp.lib urlmon.lib shell32.lib dwmapi.lib ole32.lib ^
   /link /SUBSYSTEM:WINDOWS
```

## Использование

1. Запусти `WinHub.exe`
2. Найди программу через поиск или категорию
3. Нажми **Установить** — файл скачается в `%TEMP%\WinHub\downloads\`
4. Откроется стандартный установщик программы

## Добавить свою программу

Открой `src/catalog.h` и добавь запись:

```cpp
{L"id", L"Название", L"Категория",
 L"Описание программы",
 L"https://url-to-download.exe",
 L"installer.exe", L"/S"},  // /S = тихая установка (если поддерживается)
```

Пересобери проект.

## Архитектура

```
WinHub/
├── src/
│   ├── main.cpp      # GUI (Win32)
│   ├── catalog.h     # Каталог программ
│   ├── download.cpp  # Загрузка через WinHTTP
│   └── download.h
├── build.bat
└── README.md
```

- **GUI**: Win32 API, ListView, тёмная тема через DWM
- **Загрузка**: WinHTTP (встроен в Windows)
- **Установка**: ShellExecute → стандартный установщик

## Безопасность

- Скачивание только по прямым HTTPS-ссылкам из каталога
- Файлы сохраняются во временную папку пользователя
- Проверяй URL перед добавлением в каталог
- Для production добавь проверку SHA256 хешей
